# Arrays
 A fixed-size, contiguous collection of elements of the same type, accessed by index.
## Time Complexicity
Accessing: O(1)
Insert: O(1)/O(N)
Delete: O(N)/O(1)
Traverse: O(N)
Search: a) Linear: O(N)
        b) Binary: O(logN)
Update: O(1)
## Benefits 
1) Random Access O(1)
2) Cache Friendly
3) ease of sorting
4) Implement other Data Structure
## Limitations 
1) Fixed Size
2) O(N) - Insert/ Delete Ineffiecient for Frequent modifications
## When to USE
1) Implement Other Data Structure
2) Caching
3) Tree visite nodes
4) Mathematical
5) Coding Patterns
